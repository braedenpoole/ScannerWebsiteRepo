About

- This is my market scanner project in which I scan and pull crypto and stock data from multiple api's and show them in a visually appealing and easy to absorb format.

- Most stock websites are cluttered with various ads and unnecessary statistics that halt one from understanding the status of the market and various sectors in a quick way.

- My goal in this project was to learn how to make a website using Html, CSS, and JavaScript to then pull and analyze data from various api's.

- Throughout this project I used the CoinGecko public Api for my crypto data and the finnhub private api for my stock data. Both the CoinGecko api and the finnhub api had many rate limiting hurdles I had to manage, so throughout this project I learned how to manage api calls on the back end to prevent data throttling issues on the front end.
